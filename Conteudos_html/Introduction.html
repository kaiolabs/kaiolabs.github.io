    • Introdução HTML

    HTML é a linguagem de marcação padrão para a criação de páginas da web.

    HTML

  • HTML consiste em uma série de elementos.
  • HTML significa Hyper Text Markup Language.
  • HTML descreve a estrutura de uma página da web.
  • Os elementos HTML informam ao navegador como exibir o conteúdo.
  • HTML é a linguagem de marcação padrão para a criação de páginas da web.
  • Os elementos HTML identificam partes de conteúdo como "isto é um título", "isto é um parágrafo", "isto é um link", etc.

    • Iniciando um arquivo em HTML 5

    Exemplo

    <!DOCTYPE html >

    <html lang = "pt-br">

    <head>

    <meta charset = "UTF-8">

    <meta http-equiv = "X-UA-Compatible" content = "IE=edge">

    <meta name = "viewport" content = "width=device-width, initial-scale=1.0">

    <title> Titulo da pagina </title >

    </head>

    <body>

    <h1> Olá, Mundo </h1 >

    </body>

    </html>

    Não se preocupe com todo esse código, pois, caso esteja utilizando o Visual Studio Code ele fará automática esse código para você, basta digitar "html:5" e aperta enter.

    Detalhando o código.

  • O <!DOCTYPE html> é a declaração que define o documento como HTML5.
  • O <html> é o elemento raiz de uma página HTML.
  • O <head> é o elemento que contém as metainformações sobre a página HTML.
  • O <meta> define qualquer informação de metadados que não podem ser definidos por outros elementos HTML.
  • O <title> especifica um título para a página HTML, que é mostrado na barra de título do navegador ou na guia da página.
  • O <body> é o elemento que define o corpo do documento e é um contêiner para todo o conteúdo visível da página.
  • O <h1> é um título de nível 1.

    • O que é uma tag HTML ?

    A expressão “tag” significa literalmente “etiqueta”. Uma etiqueta é usada para marcar ou identificar algo, como a etiqueta de preços no mercado por exemplo.

    No HTML5, as tags tem o mesmo objetivo: marcar elementos. Ou seja, determinar onde um elemento começa e onde ele termina. Isso é muito apropriado, afinal “HTML” significa Linguagem de Marcação de Hipertexto.

    Há dois tipos de tags:

    1) As que precisam de fechamento:

    <p> Parágrafo em html </p >

    <h1> Título em html </h1 >

    <li> Lista em html </li >

    2) As que se fecham sozinhas (self-closing):

    <meta>

    <link>

    <img>

    <br>

    O que é um elemento HTML ?

    Um elemento HTML é definido por uma tag de inicial, algum conteúdo e uma tag final:

    <tagInicial> Conteúdo </tagFinal >

    Na prática:

    <h1> Olá, Mundo </h1 >


    AVISO: Alguns elementos HTML não têm conteúdo (como o elemento <br> ). Esses elementos são chamados de elementos vazios. Os elementos vazios não possuem uma tag de finalização!
