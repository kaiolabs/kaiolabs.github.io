    • Elementos HTML

    Um elemento HTML é definido por uma tag inicial, algum conteúdo e uma tag final.

    O elemento HTML é tudo, desde a tag inicial até a tag final:


    Exemplo:

    <tagInicial> Conteúdo que sera exibido </tagFinal >

    Na prática:

    <p> Olá, Mundo </p >

    Elementos básicos

    Elementos básicos são o que constituem a base de todo documento HTML. Se vê esses elementos no código fonte de todas as páginas da web, logo em seguida da declaração de doctype, que está na primeira linha da página.

    Demostração:

    <!DOCTYPE html >

    <html>

    <head>

    <title> Titulo da pagina </title >

    </head>

    <body>

    <h1> Olá, Mundo </h1 >

    </body>

    </html>

    O elemento <html> representa a raiz de um HTML ou XHTML documento. Todos os outros elementos devem ser descendentes desse elemento.

    Então, dentro do elemento <html>, há um elemento chamado <body>:

    <html>

    <body>

    <h1> Olá, Mundo </h1 >

    <p> Meu parágrafo </p >

    </body>

    </html>

    O <body>, é o elemento que define o corpo do documento.

    Ele possui uma tag inicial <body> e uma tag final </body>.

    Então, dentro do <body> existem dois outros elementos: <h1> e <p>:

    <body>

    <h1> Olá, Mundo </h1 >

    <p> Meu parágrafo </p >

    </body>

    O elemento <h1> define um título.

    Ele tem uma tag de início <h1> e uma tag de fim </h1> :

    <h1> Olá, Mundo </h1 >

    O elemento <p> define um parágrafo.

    Ele tem uma tag de início <p> e uma tag de fim </p> :

    <p> Meu parágrafo </p >

    Elementos vazios

    Os elementos HTML sem conteúdo são chamados de elementos vazios.

    A tag <br> define uma quebra de linha e é um elemento vazio sem uma tag de fechamento:

    <p> Este é um parágrafo <br> com uma quebra de linha. </p >

    HTML não diferencia maiúsculas de minúsculas

    As tags HTML não diferenciam maiúsculas de minúsculas: <P> significa o mesmo que <p>.

    O padrão HTML não requer tags em minúsculas, mas o W3C recomenda minúsculas em HTML e exige minúsculas para tipos de documentos mais restritos como XHTML.
